    How does it work?

    This demo detects whether a comment is spam using a trained model.

    Enter a comment and click "Post Comment" to see if it's classified as spam or not. The model is trained on a dataset of YouTube comments, and it identifies common patterns in spam comments, such as excessive use of certain words, links, or phrases.
